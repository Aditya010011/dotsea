

# Dotsea – Premium Cyber-Futuristic Website

## Design System
- **Dark theme**: Deep navy `#0A0F1C` background with electric cyan `#00F5FF` and violet `#7C3AED` neon accents
- **Glassmorphism**: Translucent cards with backdrop-blur, subtle borders, and glow effects
- **Typography**: Clean modern sans-serif, large cinematic headings, generous spacing
- **Animations**: Framer Motion for scroll reveals, hover lifts, counter animations, parallax, and magnetic button effects

## Page Sections (Single-Page, Smooth Scroll)

### 1. Sticky Navbar
- Backdrop-blur glass navbar with "Dotsea" logo
- Smooth-scroll links: Home, Services, About, Portfolio, Blog, Contact
- Glowing "Get a Quote" CTA button that scrolls to Contact

### 2. Hero Section
- Full viewport with animated tech-grid and floating particle background (CSS/canvas)
- Headline: "Crafting Intelligent Digital Futures"
- Subheadline about bespoke AI/blockchain solutions
- Two CTA buttons: "Explore Services" + "Start Your Project"
- Floating laptop/phone mockup showing a dashboard visual

### 3. Services Section
- "Next-Generation Solutions" headline
- Responsive grid (3×3 on desktop) of glassmorphic cards
- Each card: Lucide icon, title, description, hover glow + scale effect
- "Learn more" opens a modal with expanded details
- 10 services: Custom Software, AI/GenAI, Mobile Apps, Web Dev, Cloud/DevOps, Blockchain/Web3, Cybersecurity, AR/VR, Data Engineering, Low-Code/Automation

### 4. About Us Section
- Split layout: mission text on left, animated stats on right
- Counter animations: 50+ Projects, 25 Clients, 100% On-time Delivery
- Team photos carousel below stats

### 5. Portfolio Section
- Masonry-style grid with 6–8 project cards
- Hover overlay with tech tags and project name
- "View Case Study" opens modal with realistic mock details and screenshots

### 6. Testimonials Section
- Horizontal carousel (shadcn-based) with client quotes
- Avatars, names, titles, and company logos
- Auto-scroll with manual navigation

### 7. Why Dotsea Section
- Horizontal feature strip with 4 key differentiators
- Icons + titles: AI-First Approach, Agile + 24/7 Delivery, End-to-End Ownership, Future-Proof Architecture

### 8. Contact Section
- Hero-style dark section with glassmorphic form
- Fields: Name, Email, Company, Service dropdown, Message
- Glowing submit button with magnetic hover effect
- "Based in Bengaluru, India" badge with location icon

### 9. Footer
- Navigation links, social media icons
- Copyright + "Built with ❤️ in India"

## Animations & Interactions
- Scroll-triggered reveal animations on every section (fade-up, scale-in)
- Magnetic button effects on CTAs
- Parallax on hero background elements
- Counter number animation on About stats
- Card hover: scale lift + cyan glow border
- Smooth scroll navigation between sections
- Floating particles/grid animation in hero background

## Fully Responsive
- Mobile-first design with hamburger menu on mobile
- Cards stack vertically on small screens
- All sections adapt gracefully to all breakpoints
